# Тестовое задание. RestaurantMenu

## Описание
Все решение подготовлено на .net 8.0 с использование winForms для визуализации работы библиотеки. Также присутствует тестовый проект, внутри которого определен набор тестов библиотеки.

### Библиотека
Представляет из себя определение для объекта MenuMaster. Для инициализации требуется определить основное меню и, опционально, зимнее меню, а также добавить кол-во блюд на странице. Класс обладает следующим функционалом:
- получение списка блюд;
- получение кол-ва блюд;
- получение кол-ва страниц меню;
- получение первых блюд на всех страницах меню;
- получение блюд на странице;
- получение первой страницы зимнего меню.

### Форма
Задание представлено в виде проекта winForms и библиотеки. оконное приложение представляет пример использования класса MenuMaster. На форме есть возможность сформировать меню и перемещаться по его страницам. Для формирования меню необходимо обязательно определить основное меню и кол-во блюд на странице, а также в конце нажать кнопку "Сформировать меню". Ниже на форме определены кнопки, для перемещения по меню, включая переход на начало зимнего меню. 

## Точки расширения
1) хранение типов меню в словаре, где ключ - имя/тип меню, а значение - список блюд;
2) выделение отдельных классов блюда, меню.
